# C++ Internship Tasks

This repository contains beginner C++ console projects created for internship practice.

## Projects

1. Number Guessing Game
   - Generates a random number between 1–100.
   - User guesses until correct.

2. Simple Calculator
   - Performs basic arithmetic operations.

3. Tic Tac Toe Game
   - Two-player console game.

4. To-Do List
   - Add, view, and remove tasks using vectors.
